# Demo app
## Create a package.json file: 
`npm init`

## Install TypeScript
`npm install typescript --save-dev`

## Initialize TypeScript project (ts.config)
`tsc --init`
 
## Install express 
`npm install express --save`
 
## Install express types
`npm install @types/express --save-dev`
 
## Compile the app
`tsc`
 
## Run express
`node app.js`
